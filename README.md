# InsFollow

Best Tool For Increase Instagram Follower.

## Requirements
1. openssl
2. curl

## How to Install in Termux

`$ pkg up -y`

`$ pkg install openssl-tool`

`$ pkg install curl`

`$ pkg install git`

`$ git clone https://github.com/MR-ZIHAD/INSTAFLOW`

`$ cd INSTAFLOW`

`$ unzip INSTAFLOW.zip`

`$ chmod +x INSTAFLOW.sh`

`$ termux-wake-lock`

`$ bash INSTAFLOW.sh`
